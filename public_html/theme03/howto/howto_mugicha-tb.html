おいしい麦茶の作り方

煮出し麦茶

沸騰したお湯の中に麦茶のティーバッグを入れて煮出す方法です。煮出し専用のティーバッグの場合にはこの方法で作ります。

  1. やかんに水を入れて火にかけます。
  2. 沸騰したお湯1.5リットルの中にティーバッグ1包を入れます。
  3. 3～5分煮出した後、火を止めて5分ほどおいてからティーバッグを取り出します。
  4. そのままホット麦茶でどうそ。
    ※冷やす場合には、氷水を張った桶にやかんごと入れて粗熱を取り、容器に入れて冷蔵庫で保存します。

お湯出し麦茶

ティーバッグを煮出さず、お湯を使って作る方法もあります。

  1. やかんなどに水を入れて火にかけます。
  2. 沸騰したら火を止め、ティーバッグを入れます。（お湯1.5リットルに1包が目安）
  3. 1時間くらいしたらティーバッグを取り出し、別の容器に移し替えます。
  4. 冷めたら冷蔵庫に入れて保管します。
